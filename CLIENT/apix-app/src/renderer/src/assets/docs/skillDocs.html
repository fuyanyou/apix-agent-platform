什么是模型技能包？
技能包是一组预定义的功能模块或指令集合，旨在为 Agent 提供执行特定任务或操作的能力，以扩展其应用范围和提升任务完成效率。
我要上传什么格式的技能包？
需要上传一个ZIP格式的技能包。该包的根目录下必须包含一个名为 SKILL.md 的文件，且其内容需符合 Anthropic 兼容的技能包协议。上传后，服务器将自动解析并加载该技能包。请不要上传同名技能包，否则按上传时间加载 latest。
上传什么技能都可以吗？
技能包主要用于封装复杂的操作逻辑，而不会扩展模型本身的基础能力。通常，你可以将复杂的任务流程编写成代码，并将其封装在技能包中，供 Agent 在需要时直接调用。注：所有的代码将会在本地执行，请不要上传恶意代码。
一次可以开启几个技能包？
开启的技能包数量未设上限，但不建议开启多个，建议关闭与任务无关的技能包以优化 Agent 表现，同时启用过多技能包可能导致指令冲突、增加决策负担，从而降低任务执行的准确性和效率。

为什么上传技能包失败？
可能原因包括：技能包中无 SKILL.md 文件、SKILL.md 中无开头的 YAML 格式元数据。
为什么技能包加载失败？
可能原因包括：压缩包解压后非技能包文件夹（技能名/SKILL.md），请确保解压后的文件夹名与 SKILL.md 中的技能名一致。
